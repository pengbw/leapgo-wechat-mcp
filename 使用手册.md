# leapgo-wechat-mcp 使用手册

> 微信公众号 MCP 插件 · v0.2.0

---

## 📦 安装

### Step 1：解压

```bash
# Linux/macOS
leapgo-wechat-mcp-v0.2.tar.gz
cd leapgo-wechat-mcp

# Windows：使用解压缩工具解压到 D:\mcp\leapgo-wechat-mcp
# 解压后进入 wechat-mp-plugin 目录
```

### Step 2：创建虚拟环境并安装

**Windows（cmd）**
```cmd
cd D:\mcp\wechat-mp-plugin
python -m venv venv
venv\Scripts\activate
pip install -e .
```

**Windows（PowerShell）**
```powershell
cd D:\mcp\wechat-mp-plugin
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -e .
```

> ⚠️ 如果 PowerShell 提示「无法加载，因为在此系统上禁止运行脚本」，请以管理员身份打开 PowerShell，运行 `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`，然后重新激活。

**Linux/macOS**
```bash
tar -xzf leapgo-wechat-mcp-v0.2.tar.gz
cd leapgo-wechat-mcp
python -m venv venv
source venv/bin/activate
pip install -e .
```

### Step 3：配置 .env 文件

在项目根目录创建 `.env` 文件：

```env
WECHAT_APP_ID=wxdd6181c2c086c7bf
WECHAT_APP_SECRET=cc2bb****58a12
```

### Step 4：验证安装

```bash
# Linux/macOS
wechat-mp-server --help

# Windows
wechat-mp-server.exe --help
```

> 如果提示命令不存在，尝试 `python -m wechat_mp.server` 或检查虚拟环境是否激活。

## 🚀 启动 MCP Server

```bash
wechat-mp-server
```

Server 通过标准输入/输出（stdio）与 MCP Client 通信，适用于 Claude Desktop、Cursor 等支持 MCP 的 AI 工具。

## 🛠 工具清单（共 20 个）

### 1️⃣ 草稿箱（5 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `draft_create` | 新建草稿 | `title`, `author`, `content` |
| `draft_list` | 草稿列表（分页） | — |
| `draft_get` | 草稿详情 | `media_id` |
| `draft_update` | 更新草稿 | `media_id` |
| `draft_delete` | 删除草稿 | `media_id` |

#### 示例

```python
# 新建草稿
draft_create(
    title="我的第一篇文章",
    author="小跃",
    content="<p>正文内容，支持HTML</p>",
    digest="这是一篇测试文章",
    thumb_media_id="your_thumb_media_id"
)

# 列出草稿（分页）
draft_list(offset=0, count=10)

# 获取草稿详情
draft_get(media_id="草稿的media_id")

# 更新草稿
draft_update(
    media_id="草稿的media_id",
    title="新标题",
    content="新内容"
)

# 删除草稿
draft_delete(media_id="草稿的media_id")
```

---

### 2️⃣ 发布（2 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `publish_submit` | 发布草稿 | `media_id` |
| `publish_list` | 发布记录列表 | — |

```python
# 发布草稿
publish_submit(media_id="草稿的media_id")

# 查看发布记录
publish_list(offset=0, count=20)
```

### 3️⃣ 数据统计（3 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `stats_article` | 图文群发数据 | `begin_date`, `end_date` |
| `stats_user` | 用户增减数据 | `begin_date`, `end_date` |
| `stats_summary` | 数据概览 | `begin_date`, `end_date` |

> 日期格式：`YYYYMMDD`，如 `20260401`

```python
# 图文数据（阅读量、分享、收藏）
stats_article(begin_date="20260401", end_date="20260422")

# 用户增减
stats_user(begin_date="20260401", end_date="20260422")

# 数据概览
stats_summary(begin_date="20260401", end_date="20260422")
```

---

### 4️⃣ 账号 & 用户（4 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `account_info` | 自动回复配置（⚠️非公众号基本信息） | — |
| `followers_list` | 关注者列表 | — |
| `user_info` | 单个用户详情 | `openid` |
| `user_batch_info` | 批量用户信息 | `openids` |

> ⚠️ 注意：`account_info` 返回的是自动回复配置，微信 API 不提供直接获取公众号头像/昵称的接口。

```python
# 关注者列表
followers_list(next_openid=None)

# 单个用户信息（含头像、性别、地区）
user_info(openid="用户的openid")

# 批量获取（最多100个）
user_batch_info(openids=["openid1", "openid2"])
```

### 5️⃣ 素材管理（4 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `material_upload_image` | 上传图片 | `file_path` |
| `material_upload_video` | 上传视频 | `file_path`, `title` |
| `material_upload_voice` | 上传音频 | `file_path` |
| `material_list` | 素材列表 | — |

> 图片最大 2MB，视频/音频最大 10MB

```python
# 上传图片
material_upload_image(file_path="/path/to/image.jpg")

# 上传视频（需提供标题和简介）
material_upload_video(
    file_path="/path/to/video.mp4",
    title="视频标题",
    introduction="视频简介"
)

# 上传音频
material_upload_voice(file_path="/path/to/audio.mp3")

# 素材列表
material_list(material_type="image", offset=0, count=20)
```

### 6️⃣ Markdown 工具（3 个）

| 工具 | 说明 | 必填参数 |
|------|------|----------|
| `md_to_html` | Markdown → HTML | `markdown` |
| `apply_theme` | 应用微信主题样式 | `html_content`, `theme` |
| `preview_html` | 生成预览文件 | `html_content` |

> 主题选项：`default` / `minimal` / `dark`

```python
# Markdown 转 HTML
html = md_to_html(markdown="# 标题\n\n正文内容")

# 应用主题
styled_html = apply_theme(html_content=html, theme="default")

# 生成预览（返回本地文件路径）
preview_html(html_content=styled_html)
```

## ⚠️ 已知限制

1. **`account_info` 查不到公众号头像** — 微信 API 本身不支持，建议登录 mp.weixin.qq.com 后台查看
2. **视频无法以「视频卡片」形式插入草稿** — 必须手动在编辑器操作
3. **`thumb_media_id` 需要先上传图片获取** — 不能通过 URL 远程图片直接引用
4. **access_token 有效期 2 小时** — 已实现自动刷新，无需手动处理

## 🤖 在 Trae IDE 中配置

Trae 基于 VS Code，提供两种 MCP Server 配置方式：

### 方式一：打开已有项目后配置（推荐）

```
1. 打开 Trae，加载项目
2. 设置 → 搜索 "MCP" → 点击 "Edit MCP Settings(JSON)"
3. 在 settings.json 中添加以下配置：
```

```json
{
  "mcpServers": {
    "wechat_mp": {
      "command": "D:\\mcp\\leapgo-wechat-mcp\\venv\\Scripts\\python.exe",
      "args": [
        "-m",
        "wechat_mp.server"
      ],
      "env": {
        "WECHAT_APP_ID": "wxdd6181c2c086c7bf",
        "WECHAT_APP_SECRET": "cc2bb****58a12"
      }
    }
  }
}
```

> ⚠️ 注意：
> - **Linux/macOS**：路径如 `/root/.hermes/venv/bin/wechat-mp-server`
> - **Windows**：路径如 `C:\Users\你的用户名\.hermes\venv\Scripts\wechat-mp-server.exe`（注意是 `Scripts` 不是 `bin`）

### 方式二：创建专用项目

如果想在任意位置管理配置，可以创建一个专门的配置文件：

**Step 1：创建项目文件夹**
```bash
mkdir -p ~/my-mcp-servers/wechat-mp
```

**Step 2：初始化并安装**
```bash
cd ~/my-mcp-servers/wechat-mp
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install wechat-mp
```

**Step 3：在 Trae 中配置**
```json
{
  "mcpServers": {
    "wechat_mp": {
      "command": "/full/path/to/venv/bin/python",
      "args": [
        "-m",
        "wechat_mp.server"
      ],
      "env": {
        "WECHAT_APP_ID": "wxdd6181c2c086c7bf",
        "WECHAT_APP_SECRET": "cc2bb****58a12"
      }
    }
  }
}
```

### 验证配置

配置完成后，在 Trae 中打开 MCP 面板（左侧工具栏 MCP 图标），应该能看到 `wechat_mp` 服务状态为 ✅ Connected。

如果状态为 ❌ Error，检查：
1. 路径是否正确（确认 `wechat-mp-server` 文件存在且可执行）
2. Python 环境是否有 `wechat-mp` 包：`pip show wechat-mp`
3. AppID / AppSecret 是否正确

## 🔧 常见问题

**Q: 报 `40001 access_token is invalid` 怎么办？**
> 检查 `.env` 中 `WECHAT_APP_ID` 和 `WECHAT_APP_SECRET` 是否正确。

**Q: 提示 `command not found` 或找不到文件？**
> - **Linux/macOS**：确认路径是 `/root/.hermes/venv/bin/wechat-mp-server`
> - **Windows**：确认路径是 `C:\Users\你的用户名\.hermes\venv\Scripts\wechat-mp-server.exe`，注意是 `Scripts` 目录，不是 `bin`

**Q: Windows 上提示 `Scripts` 目录里没有 `wechat-mp-server.exe`？**
> 先在 Windows 上重新安装：
> ```cmd
> cd C:\Users\你的用户名\.hermes\venv\Scripts
> pip install wechat-mp
> ```
> 然后确认 `wechat-mp-server.exe` 存在。

**Q: 上传图片报 `40005`？**
> 图片大小超过 2MB，或格式不支持。请使用 jpg/png/gif，大小控制在 2MB 以内。

**Q: 发布失败报 `44002`？**
> 草稿内容为空。请先确保草稿正文（content）有内容。
